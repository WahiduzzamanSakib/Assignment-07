
# Assignment 07 – Interactive Web Project

## 📌 Short Description

This project is a simple interactive web application built using **React.js**. It demonstrates component-based UI design and state management, along with a responsive and user-friendly interface. The application is deployed using **Netlify** for live hosting and easy access from any device.

---

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Netlify (Deployment)

---

## ✨ Key Features

* **Component-based architecture** – The UI is structured into reusable React components, ensuring better organization, scalability, and maintainability.
* **Interactive user interface** – Uses React state and event handling to create a dynamic and responsive user experience.
* **Live deployment** – Hosted on Netlify for fast and reliable access from any device or browser.
